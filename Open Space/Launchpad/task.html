Description

Welcome aboard! We surfed the web and found some pictures that you can use in your project: deep space for the background, a planet with lots of volcanoes, and a rocket. Your task is to organize all these pieces into one big picture. Follow the scheme, keep to the suggested class names, and you'll easily get the work done!

Here's some helpful advice:

  1. Pay attention to the rocket element: most probably, you'll need to rotate it a bit to position it on the planet.

  2. If you're having trouble placing some element where it needs to be and you can't figure out why, try to set negative values to the properties like left and bottom.

  3. If you want to hide the bottom of the planet a bit, position the bottom a little below the space area and use the CSS property overflow: hidden to hide everything that is out of the frame.

To preview your intermediate solution hover the mouse pointer over the HTML code to show the browser icons popup, and click the icon that indicates the desired browser. Alternatively, select View | Open in Browser from the main menu and then select the desired browser from the list.

Objectives

1. Implement the given scheme.

2. Set the space background to the element with class="space".

You can use rocket, space, and mars pictures from the examples, or use your own.
